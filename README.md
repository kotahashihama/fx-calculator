# 含み損益・証拠金シュミレーター

FX の含み損益・証拠金等を俯瞰できる計算アプリです。

## デモ

https://fx-calculator.netlify.com/

## 機能一覧

- 含み損益、証拠金維持率等の計算
- 計算結果の保存・削除
- Twitter ユーザー認証
- 為替レート API の取得
- Myfxbook API の取得

## 使用技術

- 言語・・・JavaScript
- フレームワーク・・・Vue.js、Nuxt.js、Vuetify
- ライブラリ・・・axios

## バージョン

- Vue.js 2.6.10
- Nuxt.js 2.7.1
- Vuetify 1.5.5
- axios 0.18.0
